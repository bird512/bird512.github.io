公司的一堆老服务器及一些阿里云服务器经常会报空间不足的警告，需要手动清理。

## 有效命令

1. 查看硬盘使用情况

   ```
   df
   ```

   ​

2. 查找文件夹使用容量

   ```
   du -h --max-depth=1
   ```

   ​

3. 一般占据空间最大的都是一些log文件，如appach以及其它nohup log。对于已经在运行中的服务的log直接rm掉可能会导致服务出错，所以可以选用以下命令来清空。

   ```
   cat /dev/null > xxx
   ```

   ​
